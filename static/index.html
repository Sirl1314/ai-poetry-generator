📜 使用说明

AI诗词生成器是基于大模型的古典诗词创作工具，支持唐诗/宋词的智能生成，兼顾格律与意境。

🎯 使用步骤

  1. 输入创作主题（如“中秋赏月”“江南水乡”）
  2. 选择诗词类型（唐诗/宋词）
  3. 选择具体体裁（如七言绝句、江城子）
  4. 点击生成按钮，等待AI创作完成

✨ 功能特色

  • 格律合规：严格遵循唐诗/宋词的平仄、押韵规则
  • 意境优美：语言典雅，贴合主题，兼具文学性
  • 解析易懂：每首诗词附带创作思路解析
  • 历史记录：自动保存创作记录，随时回顾

📝 示例主题

  • 自然景观：春日、夏雨、秋风、冬雪、山水、星空
  • 情感表达：离别、思乡、喜悦、怀古、壮志
  • 节日节气：中秋、端午、春节、清明、重阳
